R Tip of the Day: ggiraph

Mary Phillips

What is ggiraph?

  • ggiraph is an extension of the ggplot package that allows you to make interactive graphs
  • It is designed specifically to work with ggplot, so it works well with RMarkdown, Quarto and Shiny.

How does ggiraph work?

  • Each of the geometries that already exist in ggplot get an interactive version
  • geom_point() in ggiraph is geom_point_interactive()
  • geom_bar() is geom_bar_interactive()
  • and so on…
  • Simple and straightforward if you are familiar with ggplot

So, what can ggiraph do?

Scatterplot Code

dinoplot <- dinos |>
  ggplot() +
  geom_point_interactive(aes(x = type,
             y = perc,
             color = as.factor(year),
             tooltip = unique,
             data_id = unique),
             size = 3, 
             position = "jitter", 
             hover_nearest = TRUE) +
  labs(title = "How much do you like dinosuars?",
       caption = "Data from YouGov's polls, Dinosaurs
       https://yougov.com/en-us/topics/topic/Dinosaurs",
       color = "Year",
       x = "How much do you Like or Dislike dinosaurs?",
       y = "Percentage of Answers")
girafe(ggobj = dinoplot)

ggiraph arguments

  • There are a few arguments that can be added to the ggplot aesthetics.
  • tooltip: displays when the mouse hovers over objects
  • dataid: id asscoiated with elements
  • onclick: JavaScript function executed on click

Another Example: NYC Flights

mutate(name = str_glue("EWR to {dest}, ",
"flight num {flight}"))

bar <- ggplot(
  data = flights_jan1,
  mapping = aes(
    x = reorder(flight, distance), y = distance,
    tooltip = paste(name, "<br>Dist", distance, "mi"),
    data_id = name)) +
  geom_col_interactive(fill = "skyblue") +
  coord_flip() +
  labs(
    title = "Flight Distance",
    x = "Flight Number", y = "Flight Distance in Miles")

How to make the graph interactive

It will not be interactive if you don’t use the girafe() function

The code won’t fail to run or render, it will just create a regular non-interactive ggplot.

girafe(ggobj = ____)

Just insert the plot you’ve created into the space. ggobj stands for ggobject. You could also create the plot within the function.

Combining plots

If you want to use the interactive features for multiple graphs at once, use the patchwork package before you put the plotting code into girafe()

Multiple Plots + Interactive

bar <- ggplot(
  data = flights_jan1,
  mapping = aes(
    x = reorder(flight, distance), y = distance,
    tooltip = paste(name, "<br>Dist", distance, "mi"),
    data_id = name
  )
) +
  geom_col_interactive(fill = "skyblue") +
  coord_flip() +
  labs(
    title = "Flight Distance",
    x = "Flight Number", y = "Flight Distance in Miles"
  ) +
  theme_bw()

combined_plot <- scatter + bar +
  plot_layout(ncol = 2)

interactive_plot <- girafe(ggobj = combined_plot)
